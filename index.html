Förklaring av ord inom webbutveckling

Element

En html element är det man använder för att bygga upp en html dokument. Element är i huvudsak de taggar som ligger i html dokumentet. Den är allting från start till slut tag. Den berättar för webbläsaren hur man strukturerar och tolkar en del av HTML-dokumentet så att den ser ut så som man önskar.

HTML

HTML står för Hyper Text Markup Language. Enkelt kan man säga att det är ett standardspråk för att skapa webbsidor. Genom att använda html elementer kan man skapa en html dokument, alltså en vanlig hemsida vi kan se.

ID

ID inom HTML är en webbläsare stöd som ger ett unikt ID till varje enda element man använder i sin html dokument.

Class

Det som html class gör är att den ger namn (class namn) till elementer. Det som är användbar med classer är att man kan lätt fixa till saker i css genom att använda class namnen. Alltså mindre tid och mindre jobb.

Meta

Meta är en tag som används i html dokumentet för att beskriva vad hemsidan handlar om. Meta taggen påverkar inte hur hemsidan ser ut, men den är viktigt för att målgruppen ska kunna hitta din hemsida. Den ger sökmotorn informatio om vad din hemsida handlar om. När någon sedan gör en sökning på sökmotorn så kollas din sidas innehåll och din sidas META-taggar. Och då kan man lätt göra matchningar med det man söker och med olika hemsidans innehåll.

Responsive

Responsive handlar om att använda html och css för att göra en webbsida anpassad för alla enheter. Den ändrar på hemsidans storlek på tex. texter, bilder osv automatisk beroende på det enheten man använder, om det är en mobil eller dator osv.

http

Http står för Hyper Text Transfer Protocol. Det är ett protokoll som används för att överföra data/webbsidor inom World Wide Web. Detta protokoll är det som tillåter webbläsare och servrar att kommunicera genom utbyte av data. Den är en grund för datakommunikation på internet.

https

Https är en förlängning av http. Den står för samma sak som http men med ett s som betyder secure, säkerhet. Det som https skiljer sig från http är alltså säkerhet. Https skapar en krypterad anslutning mellan en webbserver och en webbläsare. Detta behövs för att skydda allt information man anger till en webbsida.

Datakomprimering (compressed data)

Datakomprimering är processen att modifiera, koda eller konvertera datastrukturen på ett sådant sätt att den tar mindre plats på disken. Detta gör man för att minska av lagrings storleken för en eller flera data instanser eller element. Datakomprimering är också känd som källkodning eller bithastighetsreduktion.

CDN

CDN står för content delivery network (innehåll leverans nätverk). En CDN möjliggör en snabb överföring av tillgångar som behövs för att ladda innehåll som HTML-sidor, javaskriptfiler, formatmallar, bilder och videor. Över hälften av all internettrafik betjänas av CDN. En välkonfigurerad CDN kan också hjälpa till att skydda mot vanliga skadliga attacker.

Local bootstrap

Bootstrap är ett gratis öppen källkod HTML / CSS / JS bibliotek som förlättar arbetsbelastningen. Dess användningsområde är ganska brett på grund av sin infrastruktur som är kompatibel med nästan alla webbläsare och dess bibliotek som innehåller många komponenter som underlättar webbdesigners arbete. Det mest största fördelen med Bootstrap-biblioteket är att den underlättar design av webbsidor så att webbplatser kan göras lämpliga för många olika skärmupplösningar, särskilt mobila enheter.

JQuery

jQuery är ett JavaScript-bibliotek. Den är det mest populära JavaScript-biblioteket som används idag. För jquery kan vi också säga att det är ett javascript-ramverk som gör det möjligt att göra mer arbete med mindre kod. Jquery används för att påskynda och underlätta användningen av JavaScript.

Flexbox

Flexblox är en av de mycket populära CSS-layouterna som underlättar webbutvecklarnas arbete, särskilt i komplexa projekt. Flexbox hjälper till att forma webbplatsstrukturen. Det är den enklaste CSS-modellen som kan användas för att enkelt ordna och justera element på platsen horisontellt och vertikalt.

Grid

Grid kallas för de horisontella och vertikala linjerna som anses osynliga för ett visst område. Grid används för att öka den visuella uppfattningen. Det viktigaste syftet med att använda Grid-systemet är att den förlättar att organisera webbplats infrastrukturen. När du skapar webbplatsens HTML-design erbjuder den en bekväm lösning eftersom man vet hur mycket utrymme det tar.

Code snippets

Code snippet är en samling av tidigare skrivna kod. Utvecklare som håller med stora projecter bygger ett bibliotek med återanvändbara kodavsnitt som kallas då för Code snippet. Detta gör man för att kunna använda koden i framtida utvecklingar.
